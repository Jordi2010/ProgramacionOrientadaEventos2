Ejercicio a Desarrollar
Crear un programa que permita convertir de grados celsius a fahrenheit, viceversa, de kelvin a celsius, viceversa y de kelvin a fahrenheit y viceversa. Para realizar este ejercicio debe crear una biblioteca de clases, en dicha biblioteca debe crear una clase principal desde donde van a heredar las clases que se usarán para calcular las conversiones.
Puede utilizar Tab Control o múltiples formularios para resolver el ejercicio.
